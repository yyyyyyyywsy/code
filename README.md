## 代码总结格式

每个代码文件都会生成对应的 `_SUMMARY.md` 文件，包含:

1. **功能概述** - 一句话说明代码做什么
2. **核心逻辑** - 主要算法、流程、数据结构
3. **关键函数表** - 名称 | 功能 | 参数 | 返回值
4. **依赖说明** - 使用的库及其用途
5. **使用方式** - 如何运行、调用示例
6. **可优化建议** - 代码质量、性能、可读性改进建议
